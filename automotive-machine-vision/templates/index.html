How confidential computing + CCE protect this workflow

  • Remote attestation provides cryptographic evidence of platform integrity before video upload is allowed.
  • CCE policy measurements are checked by ACI so the workload must match the approved container image.
  • Policy generation uses az confcom acipolicygen --disable-stdio to block interactive shell/exec access.
  • TLS (HTTPS) protects video data while in transit from browser to container.
  • Machine vision processing runs server-side in the confidential container, not in browser JavaScript.
