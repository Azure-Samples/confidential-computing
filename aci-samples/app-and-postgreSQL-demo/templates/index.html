🛡️ Hardware Attestation & Secure Key Release

Request a remote attestation token from Microsoft Azure Attestation (MAA) to cryptographically prove this container runs inside a genuine AMD SEV-SNP Trusted Execution Environment. Then use Secure Key Release (SKR) to release an HSM-backed key that is bound to this container's security policy hash — the key can only be released to an attested, unmodified container.
